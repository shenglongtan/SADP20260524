# Skills Knowledge Base - 技能知识库

## 项目说明

这是一个用于记录和管理技能操作相关知识的知识库。当你在日常浏览中发现有价值的 skills 相关内容时，将链接发给 Claude，Claude 会自动完成以下工作：

1. **抓取内容** - 访问链接，提取核心信息
2. **结构化记录** - 按统一格式写入知识条目
3. **分类归档** - 打标签、归类，更新目录索引
4. **安装集成** - 如果内容涉及可安装的技能/工具，集成到技能库中

---

## 目录结构


## 工作流程

### 你（用户）的操作：
1. 发送链接（URL）给 Claude
2. 可选：附加一句话说明你觉得这个内容的重点在哪

### Claude 的操作：
1. 访问链接，提取内容
2. 按模板格式创建知识条目文件（`entries/XXX_标题.md`）
3. 更新 `catalog.json` 索引
4. 如果内容涉及可安装的技能/配置，直接集成到项目中
5. 提交并推送到 Git 仓库
6. 回复你一段简短摘要，确认收录完成

### 快捷指令约定：
- 发送链接 = 默认执行「收录」流程
- 发送链接 + "安装" = 收录 + 将技能安装到本地
- 发送链接 + "仅记录" = 只记录不安装
- "查询 [关键词]" = 在知识库中搜索
- "统计" = 输出知识库的分类统计
